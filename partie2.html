Exercice 1 : Manipulation de tableau simple

Écrire un algorithme qui :

  1. Crée un tableau contenant les nombres : [3, 15, 7, 22, 9, 18, 4, 12]
  2. Affiche uniquement les nombres supérieurs à 10
  3. Compte et affiche combien de nombres sont supérieurs à 10

1.

Crée un tableau contenant les nombres : [3, 15, 7, 22, 9, 18, 4, 12] 

                        
    ALGORITHME :

    table ← [3, 15, 7, 22, 9, 18, 4, 12] 
                        
                    
                        
    JavaScript : 

    const table = [3, 15, 7, 22, 9, 18, 4, 12]

2.

Affiche uniquement les nombres supérieurs à 10 

                        
    ALGORITHME :

    POUR i de 0 à taille de table - 1
        SI table[i] > 10
            AFFICHER table[i]
        FIN SI
    FIN POUR
                        
                    
                        
    JavaScript : 

    // Avec la boucle for :
    for (let i = 0; i < table.length; i++) {
        if (table[i] > 10) {
            console.log(table[i])
        }  
    };

    // Avec la boucle for each :
    table.forEach(element => {
        if (element > 10) {
            console.log(element)
        }
    });

Exercice 3 : Tableau multidimensionnel

Un tableau représente les notes de 3 élèves sur 4 matières

                        
    notes ← [ 
        [12, 15, 10, 14],  // Élève 1 
        [8, 11, 9, 13],    // Élève 2 
        [16, 18, 17, 19]   // Élève 3 
    ]
  1. Écrire un algorithme qui affiche toutes les notes avec le format : 
    2.                         
    Élève 1 : 12, 15, 10, 14 
    Élève 2 : 8, 11, 9, 13 
    Élève 3 : 16, 18, 17, 19
  2. Écrire une fonction calculerMoyenneEleve(tableau, numeroEleve) qui :
    • Prend en paramètre le tableau de notes et le numéro de l'élève (0, 1, ou 2)
    • Calcule la moyenne de cet élève
    • Retourne cette moyenne
  3. Utiliser cette fonction dans un algorithme principal pour :
    • Afficher la moyenne de chaque élève
    • Identifier et afficher quel élève a la meilleure moyenne

1. Affichage

Écrire un algorithme qui affiche toutes les notes avec le format : 

                        
    FORMAT :

    Élève 1 : 12, 15, 10, 14 
    Élève 2 : 8, 11, 9, 13 
    Élève 3 : 16, 18, 17, 19 
                        
                    
                        
    ALGORITHME : 

    notes ← [ 
        [12, 15, 10, 14],  
        [8, 11, 9, 13],   
        [16, 18, 17, 19]  
    ]

    POUR i de 0 à taille de notes - 1
        note ← notes[i][0];
        POUR j de 1 à taille de notes[i] - 1
            note ← note + ", " + notes[i][j]
        FIN POUR
        AFFICHER "Elève " + (i + 1) + " : " + note"
    FIN POUR
                        
                    
                        
    JavaScript : 

    const notes = [ 
        [12, 15, 10, 14],
        [8, 11, 9, 13],
        [16, 18, 17, 19]
    ]
    
    let note = 0;

    for (let i = 0; i < notes.length; i++) {
        note = notes[i][0];
        for (let j = 1; j < notes[i].length; j++) {
            note = `${note}, ${notes[i][j]}` 
        }
        console.log("Elève " + (i + 1) + " : " + note);
    }

2. Fonction de calcul

Écrire une fonction calculerMoyenneEleve(tableau, numeroEleve) qui :

  • Prend en paramètre le tableau de notes et le numéro de l'élève (0, 1, ou 2)
  • Calcule la moyenne de cet élève
  • Retourne cette moyenne
                        
    ALGORITHME :

    FONCTION calculerMoyenneEleve(tableau, numeroEleve)
        nombreNotes ← 0
        sommeNotes ← 0
        POUR j de 0 à taille de tableau - 1
            nombreNotes ← nombreNotes + 1
            sommeNotes ← sommeNotes + tableau[j]
        FIN POUR
        moyenneEleve ← sommeNotes / nombreNotes
        SI 0 <= moyenneEleve <= 20
            RETOURNER moyenneEleve
        SINON SI
            RETOURNER "Erreur : Notes invalides"
        FIN SI
    FIN FONCTION
                        
                    
                        
    JavaScript : 

    function calculerMoyenneEleve(tableau, numeroEleve) {
        let sommeNotes = 0;
        let nombreNotes = 0;
        for (let j = 0; j < tableau.length; j++) {
            nombreNotes = nombreNotes + 1;
            sommeNotes = sommeNotes + tableau[j];
        }
        let moyenneEleve = sommeNotes / nombreNotes;
        if (moyenneEleve >= 0 && moyenneEleve <= 20) {
            return Math.round(moyenneEleve *100) / 100;
        } else {
            return 'Erreur : Notes invalides'
        }
    }

3. Utilisation

Utiliser cette fonction dans un algorithme principal pour :

  • Afficher la moyenne de chaque élève
  • Identifier et afficher quel élève a la meilleure moyenne 
                        
    ALGORITHME :
    
    AFFICHER "La moyenne de l'élève 1 est de " + calculerMoyenneEleve(notes[0], 0)
    AFFICHER "La moyenne de l'élève 2 est de " + calculerMoyenneEleve(notes[1], 1)
    AFFICHER "La moyenne de l'élève 3 est de " + calculerMoyenneEleve(notes[2], 2)

    meilleureMoyenne ← 0
    numeroEleve ← 0

    POUR i de 0 à taille notes
        SI calculerMoyenneEleve(notes[i], i) > meilleureMoyenne
            meilleureMoyenne ← calculerMoyenneEleve(notes[i], i)
        FIN SI
    FIN POUR

    AFFICHER "La meilleure moyenne de la classe est de " + meilleureMoyenne + " obtenue par l'élève " + numeroEleve
                        
                    
                        
    JavaScript :

    console.log("La moyenne de l'élève 1 est de " + calculerMoyenneEleve(notes[0], 0));
    console.log("La moyenne de l'élève 2 est de " + calculerMoyenneEleve(notes[1], 1));
    console.log("La moyenne de l'élève 3 est de " + calculerMoyenneEleve(notes[2], 2));

    let meilleureMoyenne = 0;
    let numeroEleve = 0;

    for (let i = 0; i < notes.length; i++) {
        if (calculerMoyenneEleve(notes[i], i) > meilleureMoyenne) {
            meilleureMoyenne = calculerMoyenneEleve(notes[i], i)
            numeroEleve = i + 1;
        }
    };

    console.log(`La meilleure moyenne de la classe est de ${meilleureMoyenne} obtenue par l'élève ${numeroEleve}`);

Exercice 4 : Algorithme de tri

Soit le tableau : [64, 34, 25, 12, 22, 11, 90]

Partie 1 : Algorithme complet

Écrire l'algorithme complet du tri à bulles (bubble sort)

Partie 2 : Traçage

Tracer les 2 premiers passages de l'algorithme sur ce tableau :

  • Indiquer toutes les comparaisons effectuées
  • Indiquer tous les échanges réalisés
  • Montrer l'état du tableau après chaque passage

Partie 1 : Algorithme complet

Écrire l'algorithme complet du tri à bulles (bubble sort) 

                        
    ALGORITHME :

    tableau ← [64, 34, 25, 12, 22, 11, 90]

    POUR i de 0 à taille de tableau - 2
        tri ← vrai
        POUR j de 0 à taille de tableau - 2 - i
            SI tableau[j] > tableau[j + 1]
                temp ← tableau[j]
                tableau[j] ← tableau[j + 1]
                tableau[j + 1] ← temp
                tableau ← faux
            FIN SI
        FIN POUR
        SI tri est vrai
            SORTIR DE LA BOUCLE
        FIN SI
    FIN POUR
                        
                    
                        
    JavaScript : 

    const tableEx4 = [61, 34, 25, 12, 22, 11, 90];

    for(let i = 0; i < tableEx4.length - 1; i++) {
        let tri = true;
        for(let j = 0; j < tableEx4.length - 1 - i; j++) {
            if(tableEx4[j] > tableEx4[j+1]) {
                let temp = tableEx4[j];
                tableEx4[j] = tableEx4[j+1];
                tableEx4[j+1] = temp;
                tri = false;
            }
        }
        if(tri) {
            break;
        }
    };
