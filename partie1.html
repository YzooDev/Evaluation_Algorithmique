Partie 1 : Questions théoriques

1. Qu'est ce qu'un algorithme ?

Un algorithme est une suite d'instructions logiques, ordonnées et finies, permettant de résoudre un problème ou d'atteindre un résultat précis.

En informatique, un algorithme est une séquence d'opérations qui se termine à un moment précis et produit un résultat. Il constitue la partie mathématique de la programmation, en exploitant la puissance de calcul des ordinateurs pour résoudre des problèmes. À la base, un ordinateur est essentiellement une machine à calculer qui nous permet de faire des opérations mathématiques, et un algorithme se présente comme une série d'opérations mathématiques qui aboutissent à un résultat concret.

Un algorithme :

  • Suit un ordre précis
  • Commence et se termine (il n'est pas infini)
  • Peut être compris sans ordinateur
  • Ne dépend d'aucun langage de programmation

En développement web et mobile, on pense d'abord à l'algorithme, puis on écrit le code.

2. Qu'est-ce qu'une variable ?

Une variable est un espace de mémoire qui permet de stocker une information afin de la réutiliser ou de la modifier plus tard.

En informatique, les variables sont des espaces mémoire "élémentaires" qui associent un nom (l'identifiant) à une valeur. Dans la plupart des langages et notamment les plus courants, les variables peuvent changer de valeur au cours du temps (dynamique). Dans les langages de certains paradigmes, notamment la programmation fonctionnelle, leur valeur est au contraire figée dans le temps (statique). Les variables sont utilisées par un programme informatique pour mémoriser certaines données nécessaires au fonctionnement du programme.

On peut voir une variable comme :

  • Une boîte
  • Une étiquette collée sur une valeur
  • Un casier avec un nom

Chaque variale possède :

  • Un nom
  • Une valeur
  • Un type

3. Qu'est-ce qu'une fonction ?

Une fonction est un sous-algorithme contenant un bloc d'instructions qui réalise une tâche précise. Pour celà, elle peut recevoir des données entrée et retourner un résultat.

Lorsque certaines actions sont répétées plusieurs fois dans un algorithme, il est préférable d'utiliser des fonctions pour :

  • Les isoler
  • Les réutiliser plutôt que de les réécrire

Cela permet :

  • D'éviter les répétitions
  • De réduire les erreurs
  • De rendre l'algorithme plus lisible et maintenable

Quelle est la différence entre paramètre et valeur de retour ?

Une valeur de retour est le résultat de l'exécution de la fonction. Les paramètres sont les entrées d'une fonction nécessaires à l'exécution et à l'obtention d'un résultat.

4. Qu'est-ce qu'un tableau ?

Un tableau est une collection de données, stockées dans une seule structure.

Chaque élément du tableau :

  • A une valeur
  • Possède une position, appelée index

Un tableau permet de :

  • Organiser les données
  • Les parcourir facilement
  • Appliquer des traitements (calculs, recherches, affichage)

Analysez l'algorithme ci-dessous, et identifiez les 2 erreurs présentes. Puis, proposer une correction. 

                        
    ALGORITHME calculerMoyenne 

    notes ← [12, 15, 8] 
    somme ← 0 
    POUR i de 1 à 3 
        somme ← notes[i] 
    FIN POUR 
    moyenne ← somme / 3 
    AFFICHER moyenne 
    FIN 
                        
                    
                        
    ALGORITHME calculerMoyenne  

    notes ← [12, 15, 8] 
    somme ← 0 
    POUR i de 0 à 2
        somme ← somme + notes[i] 
    FIN POUR 
    moyenne ← somme / 3 
    AFFICHER moyenne 
    FIN

    FONCTION estPair(nombre) 
        SI ____________ 
            RETOURNER vrai 
        SINON 
            RETOURNER faux 
        FIN SI 
    FIN FONCTION 
                        
                    
                        
    FONCTION estPair(nombre) 
        SI modulo 2 = O
            RETOURNER vrai 
        SINON 
            RETOURNER faux 
        FIN SI 
    FIN FONCTION

C'est une fonction qui retourne en résultat si le nombre en paramètre est pair (vrai) ou pas (faux).
