Partie 3 : Mini-projet

Système de gestion de bibliothèque

Contexte

Vous devez créer un système simplifié de gestion de bibliothèque.

Données

Un tableau 2D représente les livres de la bibliothèque :

                        
    livres ← [ 
        ["1984", "Orwell", 1949, vrai], 
        ["Le Seigneur des Anneaux", "Tolkien", 1954, faux], 
        ["Harry Potter", "Rowling", 1997, vrai], 
        ["Fondation", "Asimov", 1951, vrai] 
    ]

Structure : [titre, auteur, année, disponible]

  • disponible = vrai si le livre peut être emprunté
  • disponible = faux si le livre est déjà emprunté

Fonctionnalités à implémenter

1. Fonction afficherLivres(tableau)

2. Fonction rechercherLivre(tableau, titre)

3. Fonction emprunterLivre(tableau, position)

4. Fonction trierParAnnee(tableau)

Algorithme principal attendu

Écrire un algorithme principal qui :

  1. Affiche tous les livres
  2. Recherche un livre spécifique (titre au choix)
  3. Si trouvé, tente de l'emprunter
  4. Trie les livres par année
  5. Affiche à nouveau tous les livres (dans l'ordre trié)

Exemple de déroulement :

                        
    === Catalogue de la bibliothèque === 
    1. 1984 par Orwell (1949) - Disponible 
    2. Le Seigneur des Anneaux par Tolkien (1954) - Emprunté 
    3. Harry Potter par Rowling (1997) - Disponible 
    4. Fondation par Asimov (1951) - Disponible 
    
    Recherche du livre : Harry Potter 
    Livre trouvé à la position 2 
    Emprunt réussi : Harry Potter 
    
    Tri des livres par année... 
    
    === Catalogue trié par année === 
    1. 1984 par Orwell (1949) - Disponible 
    2. Fondation par Asimov (1951) - Disponible 
    3. Le Seigneur des Anneaux par Tolkien (1954) - Emprunté 
    4. Harry Potter par Rowling (1997) - Emprunté

1. Fonction afficherLivres(tableau)

Affiche tous les livres avec leur statut.

Format attendu :

                        
    === Catalogue de la bibliothèque === 
    1. 1984 par Orwell (1949) - Disponible 
    2. Le Seigneur des Anneaux par Tolkien (1954) - Emprunté 
    3. Harry Potter par Rowling (1997) - Disponible 
    4. Fondation par Asimov (1951) - Disponible
                        
                    
                        
    ALGORITHME :
    
    FONCTION afficherLivres(tableau)
        AFFICHER "=== Catalogue de la bibliothèque ==="
        POUR i de 0 à taille de tableau - 1
            SI tableau[i][3] est vrai
                AFFICHER i + 1 + ". " + tableau[i][0] + " par " + tableau[i][1] + " (" + tableau[i][2] + ") - Disponible"
            SINON
                AFFICHER i + 1 + ". " + tableau[i][0] + " par " + tableau[i][1] + " (" + tableau[i][2] + ") - Emprunté"
            FIN SI         
        FIN POUR
    FIN FONCTION
                        
                    
                        
    JavaScript :
    
    function afficherLivres(tableau) {
        console.log('=== Catalogue de la bibliothèque ===');
        for (let i = 0; i < tableau.length; i++) {
            if (tableau[i][3] === true) {
                console.log(i + 1 + ". " + tableau[i][0] + " par " + tableau[i][1] + " (" + tableau[i][2] + ") - Disponible");
            } else {
                console.log(i + 1 + ". " + tableau[i][0] + " par " + tableau[i][1] + " (" + tableau[i][2] + ") - Emprunté");
            }
        }
    };

2. Fonction rechercherLivre(tableau, titre)

Cherche un livre par son titre (recherche séquentielle).

Paramètres :

  • tableau : le tableau des livres
  • titre : le titre recherché

Retour :

  • L'index du livre si trouvé
  • (-1) si non trouvé
                        
    ALGORITHME :

    FONCTION rechercherLivre(tableau, titre)
        POUR i de 0 à taille de tableau - 1 
            SI tableau[i][0] = titre
                RETOURNER i
            FIN SI
            RETOURNER -1
        FIN POUR
    FIN FONCTION 
                        
                    
                        
    JavaScript :

    function rechercherLivre(tableau, titre) {
        for (let i = 0; i < tableau.length; i++) {
            if (tableau[i][0] === titre) {
                return i
            }
        }
        return -1;
    };

3. Fonction emprunterLivre(tableau, position)

Gère l'emprunt d'un livre.

Paramètres :

  • tableau : le tableau des livres
  • position : l'index du livre à emprunter

Comportement :

  • Vérifier si le livre existe (position valide)
  • Si le livre est disponible :

    • - Changer le statut à faux

    - Afficher "Emprunt réussi : [titre]"

  • Si le livre est déjà emprunté :

    • - Afficher "Ce livre est déjà emprunté"

  • Si la position est invalide :

    • - Afficher "Livre non trouvé"

                        
    ALGORITHME :

    FONCTION emprunterLivre(tableau, position)
        SI tableau[position] est vrai //(existe)
            SI rechercherLivre(tableau, tableau[position][0]) != -1
                SI tableau[position][3] est vrai
                    AFFICHER "Emprunt réussi : " + tableau[position][0]
                    RETOURNER tableau[position][3] ← faux
                FIN SI
                AFFICHER "Ce livre est déjà emprunté"
                FIN DE FONCTION
            FIN SI
        FIN SI
        AFFICHER "Livre non trouvé"
    FIN FONCTION 
                        
                    
                        
    JavaScript :

    function emprunterLivre(tableau, position) {
        if (tableau[position]) {
            if (rechercherLivre(tableau, tableau[position][0]) != -1) {
                if (tableau[position][3]) {
                    console.log("Emprunt réussi : " + tableau[position][0]);
                    return tableau[position][3] = false;
                }
            return console.log("Ce livre est déjà emprunté");   
            }
        }
        return console.log("Livre non trouvé");
    };

4. Fonction trierParAnnee(tableau)

Trie les livres par année de publication (ordre croissant).

Important : Utiliser le tri par sélection ou le tri à bulles

Paramètre :

  • tableau : le tableau des livres

Comportement :

  • Le tri doit conserver toutes les informations du livre (titre, auteur, année, disponible)
  • L'ordre final doit être croissant selon l'année
                        
    ALGORITHME :

    FONCTION trierParAnnee(tableau)
        POUR i de 0 à taille de tableau - 2
            index ← i
            POUR j de i + 1 à taille de tableau - 1
                SI tableau[j][2] < tableau[index][2]
                    index ← j
                FIN SI
            FIN POUR
            SI index != i
                temp ← tableau[index]
                tableau[index] ← tableau[i]
                tableau[i] ← temp
            FIN SI 
        FIN POUR
    FIN FONCTION 
                        
                    
                        
    JavaScript :

    function trierParAnnee(tableau) {
        for (let i = 0; i < tableau.length - 1; i++) {
            let index = i;
            for (let j = i + 1; j < tableau.length; j++) {
                if (tableau[j][2] < tableau[index][2]) {
                    index = j
                }
            }
            if (index !== i) {
                let temp = tableau[index];
                tableau[index] = tableau[i];
                tableau[i] = temp
            }
        } 
    };
